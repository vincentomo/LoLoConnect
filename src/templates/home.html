Data Structures

In computer science, a data structure is a data organization, management and storage format that enables efficient access and modification. More precisely, a data structure is a collection of data values, the relationships among them, and the functions or operations that can be applied to the data.

Good Data Structures knowledge is key and fundamental in both Computing theory and Software Development world. For Software to be well functional and be remain competent it has to operational on the right Data Structures. Software is a key part of human day-to-life and has to be developed using the best practises and Design principles to meet human demand. Data structures forms a key part in this.
